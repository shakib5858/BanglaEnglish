বেতন Override

Auto-Approval

Collection → AuditLog → বেতন পরিশোধ → AuditLog → অগ্রিম → AuditLog

  • HMAC-SHA256 password pepper (auth.service.ts)
  • ভূমিকা অনুযায়ী rate limiting (Admin থেকে)
  • সব mutation এ CSRF সুরক্ষা (csrf.ts)
  • প্রতিটি endpoint এ input sanitization
  • Gateway credentials AES-256 encrypted
  • Immutable audit logs (UPDATE/DELETE নেই)
  • Multi-tenant isolation (businessId)
  • V1 API তে request signing
