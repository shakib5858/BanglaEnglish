Key Metrics ও KPIs

সঠিক Metrics track করা business growth এর জন্য অপরিহার্য।

Traffic Metrics:

  • Users: Unique visitors
  • Sessions: Website visits
  • Page Views: Total pages viewed
  • Bounce Rate: Single page visits
  • Session Duration: Time spent on site

Engagement Metrics:

  • Pages per Session: Depth of engagement
  • Average Session Duration: Quality of content
  • Engagement Rate: Active user percentage
  • Scroll Depth: Content consumption
  • Event Tracking: Specific interactions

Conversion Metrics:

  • Goal Completions: Desired actions
  • Conversion Rate: Success percentage
  • Revenue: Sales generated
  • ROI: Return on investment
  • Customer Lifetime Value: Long-term value

Customer Journey Analysis

Customer Journey বুঝলে marketing optimization করা সহজ হয়।

Attribution Models:

  • First-click: First touchpoint credit
  • Last-click: Final touchpoint credit
  • Linear: Equal credit distribution
  • Time-decay: Recent touchpoints more credit
  • Data-driven: AI-based attribution

Multi-Channel Funnels:

  • Awareness: First brand interaction
  • Consideration: Research ও comparison
  • Decision: Purchase intent
  • Action: Conversion
  • Retention: Repeat engagement
Journey Optimization

Google Analytics 4 Setup

Initial Setup Process:

  1. Google Account:
    Gmail account দিয়ে analytics.google.com এ sign in
  2. Property Creation:
    "Create Property" → Website details enter
  3. Data Stream Setup:
    Web stream create করুন website URL দিয়ে
  4. Tracking Code:
    Global Site Tag (gtag.js) copy করুন
  5. Code Implementation:
    Website এর section এ code add করুন 
Google Analytics 4 Tracking Code:

Verification Steps:

  • Debug Mode: Implementation verify করুন

Goals ও Conversions Setup

Conversion Events Configuration:

Standard Events:
  • Purchase: E-commerce transactions
  • Sign_up: User registrations
  • Login: User authentication
  • Contact: Form submissions
  • Download: File downloads
Custom Events Setup:
Custom Event Tracking:

// Form submission tracking
gtag('event', 'form_submit', {
  'event_category': 'engagement',
  'event_label': 'contact_form',
  'value': 1
});

// Button click tracking
gtag('event', 'click', {
  'event_category': 'engagement',
  'event_label': 'cta_button',
  'value': 1
});

// Scroll depth tracking
gtag('event', 'scroll', {
  'event_category': 'engagement',
  'event_label': '75_percent',
  'value': 75
});
