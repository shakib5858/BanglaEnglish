Core Web Vitals কী?

Core Web Vitals হলো Google এর ৩টি গুরুত্বপূর্ণ মেট্রিক যা User Experience পরিমাপ করে।

৩টি Core Web Vitals
  • LCP (Largest Contentful Paint): ২.৫ সেকেন্ডের কম
  • FID (First Input Delay): ১০০ মিলিসেকেন্ডের কম
  • CLS (Cumulative Layout Shift): ০.১ এর কম

কেন গুরুত্বপূর্ণ?

  • Google Ranking Factor (২০২১ থেকে)
  • User Experience উন্নতি
  • Bounce Rate কমানো
  • Conversion Rate বৃদ্ধি

Mobile-First Indexing

Google এখন প্রথমে Mobile Version দেখে, তারপর Desktop।

Mobile Optimization Checklist
  • Responsive Design: সব ডিভাইসে সঠিকভাবে দেখায়
  • Fast Loading: ৩ সেকেন্ডের মধ্যে লোড
  • Touch-Friendly: বাটন ও লিঙ্ক সহজে ক্লিক করা যায়
  • Readable Text: ছোট স্ক্রিনে পড়া যায়

Technical SEO Elements

Technical SEO এর মূল উপাদানগুলো:

গুরুত্বপূর্ণ Technical Factors:

  • Site Speed: পেজ লোডিং স্পিড
  • SSL Certificate: HTTPS সিকিউরিটি
  • XML Sitemap: সার্চ ইঞ্জিনের জন্য রোডম্যাপ
  • Robots.txt: ক্রলিং নির্দেশনা
  • Schema Markup: Structured Data
  • Canonical Tags: Duplicate Content এড়ানো

Page Speed Optimization

  1. Speed Test করুন:
    Google PageSpeed Insights: pagespeed.web.dev
  2. Image Optimization:
    TinyPNG, Compressor.io দিয়ে ছবি কমপ্রেস করুন
  3. Caching Plugin ব্যবহার করুন:
    WordPress: WP Rocket, W3 Total Cache
  4. CDN সেটআপ করুন:
    Cloudflare (বিনামূল্যে) ব্যবহার করুন
সতর্কতা

অনেক plugin একসাথে ব্যবহার করলে সাইট slow হতে পারে। প্রয়োজনীয় plugin-ই রাখুন।

XML Sitemap তৈরি

WordPress এ Sitemap তৈরি:

  1. Yoast SEO Plugin:
    Yoast → General → Features → XML Sitemaps চালু করুন
  2. Sitemap URL:
    yoursite.com/sitemap_index.xml
  3. Google Search Console এ Submit:
    Sitemaps → Add a new sitemap 
Sitemap URL Examples:
yoursite.com/sitemap_index.xml
yoursite.com/post-sitemap.xml
yoursite.com/page-sitemap.xml

SSL Certificate সেটআপ

SSL Certificate এর গুরুত্ব:

  • Google Ranking Factor
  • User Trust বৃদ্ধি
  • Data Security
  • Browser Warning এড়ানো

বিনামূল্যে SSL পাওয়ার উপায়:

  1. Let's Encrypt: letsencrypt.org
  2. Cloudflare SSL: cloudflare.com
  3. Hosting Provider: অধিকাংশ হোস্টিং কোম্পানি বিনামূল্যে SSL দেয়

Technical SEO Audit Tools

বিনামূল্যে Audit Tools:

  • Google Search Console: Technical issues দেখুন
  • Google PageSpeed Insights: Speed analysis
  • GTmetrix: gtmetrix.com
  • Screaming Frog: screamingfrog.co.uk (500 URL বিনামূল্যে)

Audit Checklist:

  1. Crawl Errors চেক করুন
  2. Broken Links খুঁজুন
  3. Duplicate Content দেখুন
  4. Missing Meta Tags খুঁজুন
  5. Image Alt Text চেক করুন

শিক্ষকের জন্য নির্দেশনা

ক্লাস শুরুর আগে (৫ মিনিট):

  • একটি slow website এর example প্রস্তুত রাখুন
  • Mobile ও Desktop view এর পার্থক্য দেখানোর জন্য প্রস্তুত থাকুন

  • Speed Test Demo (১০ মিনিট): Students দের সাইট test করুন
  • Mobile vs Desktop (৫ মিনিট): Mobile-first indexing এর গুরুত্ব
  • Quick Fixes (৫ মিনিট): সহজ optimization tips

প্র্যাকটিক্যাল সেশন (১৫ মিনিট):

  • Image compression tool ব্যবহার করতে শেখান
  • XML Sitemap submit করতে সাহায্য করুন
সময় ব্যবস্থাপনা

Technical বিষয় বেশি জটিল না করে practical demonstration এ বেশি সময় দিন।

আজকের হোমওয়ার্ক

  1. Website Speed Test
    Google PageSpeed Insights দিয়ে আপনার সাইটের Mobile ও Desktop speed test করুন
  2. Core Web Vitals Check
    Search Console এ Core Web Vitals report দেখুন এবং issues note করুন
  3. Image Optimization
    আপনার সাইটের সব বড় ছবি compress করুন (TinyPNG ব্যবহার করুন)
  4. SSL Certificate Check
    আপনার সাইটে SSL আছে কিনা চেক করুন। না থাকলে activate করুন
  5. XML Sitemap Submit
    Google Search Console এ XML sitemap submit করুন
