Campaign Performance Metrics

PPC Campaign এর সফলতা পরিমাপের জন্য গুরুত্বপূর্ণ মেট্রিক্স:

মূল Performance Indicators
  • CTR (Click-Through Rate): ক্লিক রেট - ২% এর উপরে ভালো
  • CPC (Cost Per Click): প্রতি ক্লিকের খরচ
  • CPA (Cost Per Acquisition): প্রতি কাস্টমারের খরচ
  • ROAS (Return on Ad Spend): বিজ্ঞাপনের রিটার্ন
  • Quality Score: Google এর দেওয়া স্কোর (১-১০)

Performance Analysis:

  • Impressions: কতবার বিজ্ঞাপন দেখানো হয়েছে
  • Clicks: কতবার ক্লিক হয়েছে
  • Conversions: কতজন কাস্টমার হয়েছে
  • Impression Share: আপনার বিজ্ঞাপন কত% সময় দেখানো হয়

Quality Score Optimization

Quality Score আপনার বিজ্ঞাপনের মান নির্ধারণ করে এবং CPC কমায়।

Quality Score এর ৩টি উপাদান
  • Expected CTR: প্রত্যাশিত ক্লিক রেট
  • Ad Relevance: বিজ্ঞাপনের প্রাসঙ্গিকতা
  • Landing Page Experience: ল্যান্ডিং পেজের মান

Quality Score উন্নত করার উপায়:

  1. Relevant Keywords: সঠিক keywords ব্যবহার করুন
  2. Compelling Ad Copy: আকর্ষণীয় বিজ্ঞাপন লিখুন
  3. Landing Page Optimization: পেজ দ্রুত লোড ও relevant করুন
  4. Negative Keywords: অপ্রয়োজনীয় keywords বাদ দিন

A/B Testing Strategy

A/B Testing দিয়ে বিভিন্ন বিজ্ঞাপনের পারফরম্যান্স তুলনা করুন।

A/B Test করার বিষয়সমূহ:

  • Headlines: বিভিন্ন শিরোনাম টেস্ট করুন
  • Descriptions: আলাদা বিবরণ দিয়ে দেখুন
  • Call-to-Action: বিভিন্ন CTA বাটন
  • Landing Pages: আলাদা পেজ টেস্ট করুন
  • Bidding Strategies: বিভিন্ন বিডিং কৌশল
A/B Testing নিয়ম

একসাথে একটি মাত্র element টেস্ট করুন। সব কিছু একসাথে পরিবর্তন করলে কোনটা কাজ করেছে বুঝবেন না।

Campaign Monitoring Dashboard

  1. Google Ads Dashboard এ যান:
    Campaigns → Overview দেখুন
  2. Key Metrics চেক করুন:
    CTR, CPC, Conversions, Quality Score দেখুন
  3. Time Period সেট করুন:
    Last 7 days, 30 days বা custom range নির্বাচন করুন
  4. Performance Report তৈরি করুন:
    Reports → Predefined reports ব্যবহার করুন
Monitoring Schedule

দৈনিক ১৫ মিনিট campaign check করুন। সাপ্তাহিক detailed analysis করুন।

Conversion Tracking Setup

Google Ads Conversion Tracking:

  1. Conversions → Summary এ যান:
    Google Ads dashboard থেকে
  2. "+" বাটনে ক্লিক করুন:
    নতুন conversion action তৈরি করুন
  3. Conversion Type নির্বাচন:
    Website, Phone calls, App, বা Import
  4. Tracking Code Install:
    আপনার website এ tracking code যোগ করুন 
Google Ads Conversion Code Example:
<script>
gtag('event', 'conversion', {
    'send_to': 'AW-XXXXXXXXX/XXXXXX',
    'value': 1.0,
    'currency': 'BDT'
});
</script>

Campaign Optimization Steps

Weekly Optimization Checklist:

  1. Low Performing Keywords পরিবর্তন:
    CTR 1% এর কম keywords pause করুন
  2. Negative Keywords যোগ:
    Irrelevant search terms বাদ দিন
  3. Bid Adjustment:
    High performing keywords এর bid বাড়ান
  4. Ad Copy Testing:
    নতুন ad variations তৈরি করুন
  5. Landing Page Optimization:
    Page speed ও relevance উন্নত করুন

Mobile Optimization

Mobile Campaign এর জন্য বিশেষ কৌশল:

  • Mobile-Specific Ad Copy: ছোট স্ক্রিনের জন্য সংক্ষিপ্ত
  • Click-to-Call Extensions: ফোন নম্বর যোগ করুন
  • Location Extensions: ঠিকানা দেখান
  • Mobile Landing Pages: Mobile-optimized পেজ ব্যবহার করুন
Mobile Stats

বাংলাদেশে ৮০%+ searches মোবাইল থেকে হয়। Mobile optimization অবশ্যই করুন।
