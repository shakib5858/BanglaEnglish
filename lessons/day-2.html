On-Page SEO কৌশল

ওয়েবসাইটের ভিতরে SEO Optimization এর সব কৌশল

আজকের শিক্ষার লক্ষ্য

  • Title Tag এবং Meta Description অপটিমাইজ করতে পারবেন
  • Header Tags (H1, H2, H3) সঠিকভাবে ব্যবহার করতে পারবেন
  • Content Optimization এর কৌশল জানতে পারবেন
  • Internal Linking Strategy বুঝতে পারবেন

Title Tag অপটিমাইজেশন

Title Tag হলো আপনার পেজের সবচেয়ে গুরুত্বপূর্ণ On-Page SEO উপাদান।

Title Tag এর নিয়ম
  • ৫০-৬০ ক্যারেক্টারের মধ্যে রাখুন
  • প্রধান keyword শুরুতে রাখুন
  • প্রতিটি পেজের জন্য আলাদা title
  • ব্র্যান্ড নাম শেষে যোগ করুন

ভালো Title Tag এর উদাহরণ:

<title>ডিজিটাল মার্কেটিং কোর্স - ৩০ দিনে এক্সপার্ট | FSP Team</title>

Meta Description

Meta Description Google এ আপনার পেজের বিবরণ দেখায়।

Meta Description টিপস
  • ১৫০-১৬০ ক্যারেক্টারের মধ্যে
  • আকর্ষণীয় এবং তথ্যবহুল
  • Call-to-action যোগ করুন
  • Primary keyword অন্তর্ভুক্ত করুন
<meta name="description" content="৩০ দিনে ডিজিটাল মার্কেটিং শিখুন। SEO, Social Media, Google Ads - সব কিছু বাংলায়। আজই শুরু করুন!">

Header Tags (H1, H2, H3)

Header Tags আপনার কন্টেন্টের কাঠামো তৈরি করে।

Header Tags এর সঠিক ব্যবহার:

  • H1: প্রতি পেজে একটি, প্রধান শিরোনাম
  • H2: মূল বিভাগের শিরোনাম
  • H3: উপ-বিভাগের শিরোনাম
  • H4-H6: আরও ছোট বিভাগের জন্য

WordPress এ On-Page SEO

  1. Yoast SEO Plugin ইনস্টল করুন:
    WordPress Admin → Plugins → Add New → "Yoast SEO" সার্চ করুন
  2. Post/Page এডিট করুন:
    নিচে Yoast SEO বক্স দেখবেন
  3. Focus Keyword সেট করুন:
    আপনার টার্গেট keyword লিখুন
  4. Title ও Meta Description লিখুন:
    Google Preview দেখে অপটিমাইজ করুন
সতর্কতা

Keyword stuffing করবেন না। স্বাভাবিকভাবে keyword ব্যবহার করুন।

Internal Linking Strategy

Internal Link তৈরির নিয়ম:

  1. Relevant Content Link করুন:
    সম্পর্কিত পোস্টের সাথে লিঙ্ক করুন
  2. Anchor Text অপটিমাইজ করুন:
    "এখানে ক্লিক করুন" এর বদলে বর্ণনামূলক টেক্সট ব্যবহার করুন
  3. Deep Linking করুন:
    শুধু হোমপেজ নয়, অন্যান্য গুরুত্বপূর্ণ পেজেও লিঙ্ক করুন 
ভুল: <a href="/seo-guide">এখানে ক্লিক করুন</a>
সঠিক: <a href="/seo-guide">সম্পূর্ণ SEO গাইড</a>

Image Optimization

ছবি অপটিমাইজেশনের ধাপ:

  1. Alt Text যোগ করুন:
    প্রতিটি ছবিতে বর্ণনামূলক alt text দিন
  2. File Name অপটিমাইজ করুন:
    "IMG_001.jpg" এর বদলে "digital-marketing-course.jpg" ব্যবহার করুন
  3. File Size কমান:
    TinyPNG বা Compressor.io ব্যবহার করুন

শিক্ষকের জন্য নির্দেশনা

ক্লাস শুরুর আগে (৫ মিনিট):

  • একটি WordPress site খুলে রাখুন
  • Yoast SEO plugin ইনস্টল করে রাখুন
  • Google এ কিছু search result দেখান

মূল পাঠ (৩৫ মিনিট):

  • Title Tag ব্যাখ্যা (১০ মিনিট): Google search result এ title কীভাবে দেখায় তা দেখান
  • Meta Description (১০ মিনিট): বিভিন্ন ওয়েবসাইটের meta description তুলনা করুন
  • Header Tags (১০ মিনিট): একটি blog post এর structure দেখান
  • Practical Demo (৫ মিনিট): Live WordPress এ optimization করুন

প্র্যাকটিক্যাল সেশন (১৫ মিনিট):

  • Students দের নিজেদের website/blog এ কাজ করতে দিন
  • Title ও Meta Description লিখতে সাহায্য করুন
  • Common mistakes গুলো ধরিয়ে দিন
সময় ব্যবস্থাপনা

Theory অংশে বেশি সময় না দিয়ে practical এ বেশি ফোকাস করুন। Students যেন হাতে-কলমে শিখতে পারে।

আজকের হোমওয়ার্ক

  1. ৫টি Page এর Title Tag অপটিমাইজ করুন
    আপনার website/blog এর ৫টি গুরুত্বপূর্ণ page এর title tag উন্নত করুন
  2. Meta Description লিখুন
    একই ৫টি page এর জন্য আকর্ষণীয় meta description তৈরি করুন
  3. Header Structure তৈরি করুন
    একটি blog post এ সঠিক H1, H2, H3 structure ব্যবহার করুন
  4. Internal Link যোগ করুন
    আপনার content এ কমপক্ষে ৩টি relevant internal link যোগ করুন
  5. Image Alt Text যোগ করুন
    আপনার website এর সব ছবিতে proper alt text দিন
